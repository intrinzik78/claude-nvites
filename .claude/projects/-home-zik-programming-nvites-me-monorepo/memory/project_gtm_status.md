As of 2026-04-02, the go-to-market model is an open question. Three options on the table:
1. Business consulting targeting hyper-local companies (HVAC, etc.) already spending on mailers/offline media
2. Direct sales
3. Guided self-serve

**Why:** The attribution problem (scan → session → conversion → revenue) requires integration work that contradicts pure self-serve SaaS. But the target market is local businesses, not enterprise — so the GTM must match the customer's sophistication level.

**How to apply:** Don't assume self-serve SaaS is the model. When making product/architecture decisions, consider how the choice affects all three GTM paths. The user is actively thinking through this and wants collaborative input, not just implementation.
